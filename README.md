Moviles
=======


Este repositorio contiene los apuntes del módulo "Programación multimedia y de dispositivos móviles" impartido en el IES Maestre de Calatrava durante el curso 2014-2015. Puede encontrarse un PDF con todos los apuntes en el subdirectorio ``_build\latex`` (usar el botón ``RAW`` para descargar el PDF) y un archivo HTML en ``_build\singlehtml``.
